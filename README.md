# Trabajo Recuperación RA2 - PSP

## Descripción
Este proyecto implementa el uso de hilos y corrutinas en Kotlin, comparando su comportamiento en distintos escenarios de ejecución concurrente.

## Objetivo
El objetivo es analizar las diferencias entre hilos y corrutinas mediante ejemplos prácticos:
- Ejecución secuencial vs concurrente
- Comunicación entre procesos (productor-consumidor)
- Manejo de errores

## Contenido
- Hilo básico
- Comparación secuencial vs concurrente (hilos)
- Productor-consumidor con sincronización (hilos)
- Manejo de errores (hilos)
- Corrutinas básicas
- Comparación secuencial vs concurrente (corrutinas)
- Productor-consumidor con Channel (corrutinas)
- Manejo de errores (corrutinas)

## Tecnologías utilizadas
- Kotlin
- kotlinx.coroutines

## Ejecución
1. Abrir el proyecto en IntelliJ IDEA
2. Ejecutar el archivo `Main.kt`
